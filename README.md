# 2048
2048 C version

|----|----|----|----|
|    |    |    |  16|
|----|----|----|----|
|    |   4|  32|   8|

no record, score, just simple game logic.

run unix(linux) only, no compatibility window. kbhit from vendor.

make: cc 2048.c kbhit.c

run: ./a.out 

play: h(Left), j(Down), k(Up), l(Right)

quit: q(quit)
